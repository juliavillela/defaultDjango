Variables

color and style variables are custumized and replace default bs values. witch are inherited into both the reboot and the components

gray-scale: a custom gray scale replaces the bs gray.

color-definitions: a custom rainbow replaces the bs $colors map.

theme-colors: based on gray scale and color definitions, theme-colors are replaced.

Borders, hr, shadows: graphics are also overwritten. like default border witdh and color, box-shadow marker colors and hr

Reboot

bs native reboot: is applied, but inherits from custom variables.

aditional tags: some tags whitch are not modified in bs reboot, have custom reboot properties. (like tables and sections)

classes as mixin: buttons, form inputs and some other tags, are rebooted to include more complex formatting which is based on the respective bs classes and inlcuded in the default tag as a mixin.

Basic methods

active menu link: the element of class nav-item and href property === current href is set to active.
